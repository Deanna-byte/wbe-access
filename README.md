# 百万咨询室

向世界顶尖思想家请教 - 一个问题，多位大师的智慧

## 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 配置API密钥
编辑 `.env.local`，填入你的Anthropic API密钥：
```
ANTHROPIC_API_KEY=your_actual_api_key_here
```

获取密钥：https://console.anthropic.com/

### 3. 运行
```bash
npm run dev
```

打开 http://localhost:3000
